# Assistant Bancaire Intelligent (IA)

## À propos
Ce projet est un prototype développé durant mon stage au **Digital Center d'Attijariwafa bank**. Il s'agit d'un chatbot capable de comprendre les demandes des clients grâce à l'Intelligence Artificielle Générative.

## Fonctionnalités
* **IA Générative :** Compréhension du langage naturel pour répondre aux questions complexes.
* **Interface Bancaire :** Design fidèle à la charte graphique d'Attijariwafa bank.
* **Support Client :** Réponses sur les comptes, cartes et l'application mobile.

## Technologies
* **Backend :** Python, Flask
* **Frontend :** HTML5, CSS3, JavaScript
* **IA :** Intégration d'API (DeepSeek)
